# Design Decisions

## Decision Date: 2026-05-05

## Design Aesthetic: Professional

### Color Palette
- **Primary**: Deep blue (#1a365d)
- **Secondary**: Slate gray (#4a5568)
- **Accent**: Teal (#319795)
- **Background**: Light gray (#f7fafc)
- **Card Background**: White (#ffffff)
- **Text**: Dark gray (#2d3748)

### Typography
- **Headings**: System sans-serif (clean, professional)
- **Body**: System sans-serif
- **Code**: Monospace for tech details

### Layout
- **Header**: Fixed navigation with logo and links
- **Hero**: Name, title, brief tagline
- **Projects**: Grid layout with card design
- **Footer**: Social links, copyright

### Card Design
- White background with subtle shadow
- Project name as title
- Description from GitHub
- Language badge
- Link to repository

### Responsive
- Mobile-first approach
- Grid: 1 column (mobile), 2 columns (tablet), 3 columns (desktop)

### Rationale
Professional design chosen by user. Clean, corporate-friendly aesthetic that works for DevOps/MLOps portfolios. Good contrast for readability, subtle animations for polish.
